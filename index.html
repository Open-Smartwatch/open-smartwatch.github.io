The Open-SmartWatch is an open source project created by @pauls_3d_things/p3dt.net. It consists of hardware modules, 3D printable cases and a custom operating system to build smartwatches with the goal that they can be assembled, repaired, and modified by everyone. There are currently two versions:

  • Light edition (released)
  • GPS edition (preview, work in progress)

Check out the apps to see what the watch can do!
